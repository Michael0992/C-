Go to /bin/Debug/ to execute the Game.

Maze of Monolith is a simple console game. I created it as a specialist project as part of my training as an IT specialist. 
Special contents created by me in this project are:

 - An image generator that is able to display images in the console.

- A class that deletes selected lines in the console.

- A class that creates a menu 

- A class that uses NAudio Lib and provides 2 chanels for audio.
